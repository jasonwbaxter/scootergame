# Moovers Racer - Design Specification

## Overview
Moovers Racer is a high-fidelity Android racing game inspired by Vespa parts culture, combining authentic scooter handling, deep modular customization, and a workshop-forward progression loop.

## Game Pillars

### 1. Authentic Scooter Handling
- Semi-simulation physics optimized for low-speed drifting
- Weight transfer mechanics
- Clutch and turbo engagement
- Touch and controller input support
- Predictable, skill-based handling model

### 2. Customization and Parts Economy
- **Part Types**: Rims, exhausts, body panels, paint, decals, mirrors, engines, suspension
- **Rarity System**: Common → Uncommon → Rare → Epic → Legendary
- **Stat Effects**: Engine power, handling, weight, durability
- **Visual Customization**: Materials, decals, paint layers, modular meshes

### 3. Workshop Progression
- Garage as central hub
- Part unlock progression
- 3D preview and inspection
- Stat comparison and optimization

## Gameplay Features

### Vertical Slice (Sprint 0–4)
- **Career Intro**: Story beat introducing player to workshop
- **Tutorial Race**: Learn handling basics
- **One City Circuit**: Multi-lap race (3 minutes)
- **Garage Customization**: Equip 6+ parts
- **Leaderboard Preview**: Track best lap time

### Post-Vertical Slice
- Multiple tracks with distinct aesthetics
- Career mode with sponsor contracts
- Time trials with ghost replays
- Photo mode for custom scooter captures
- Seasonal cosmetic cosmetic drops
- Asynchronous multiplayer leaderboards

## Monetization Model
- **Premium Purchase**: Full game unlock
- **Cosmetic Store**: Decals, paint colors, rim designs (no stat advantage)
- **Seasonal Content**: Limited-time cosmetics and challenges
- **No Pay-to-Win**: Performance parts only via gameplay

## Visual Identity
- **Photoreal Stylized PBR**: Worn paint, grease, polished chrome, procedural dirt and wear
- **Lighting**: Dynamic global illumination (Lumen), HDR bloom for night, cinematic depth-of-field for replays
- **Color Palette**: Workshop grays/reds, neon accents, authentic Vespa heritage colors
- **UI Style**: Industrial/parts catalog aesthetic, clean and tactile

## Audio Identity
- **Engine**: Clutch clicks, belt whine, mechanical feedback
- **Turbo**: Compressor whine, boost rush
- **Ambient**: Workshop ambience, street racing atmosphere
- **Music**: Curated blend of retro scooter vibes and modern electronic beats
