## Customization Limits & Combinations

### Parts Per Type (Maximum Equipped)
- Rims: 1 set (both wheels)
- Exhausts: 1 system
- Body Panels: 1 full kit
- Paint: 1 primary + 1 secondary + 1 tertiary
- Decals: Up to 3 individual decals (total size limit)
- Mirrors: 1 pair (both sides)
- Engine: 1 engine block
- Suspension: 1 suspension kit

### Color Coordination Guidelines
- **Harmonious**: Parts from same brand pair naturally
- **Contrast**: Neon finishes pair with matte bases
- **Premium Mixing**: Legendary parts work with any finish
- **Clash Warning**: Opposite aesthetic brands clash visually

### Stat Impact by Rarity
| Rarity | Stat Boost | Visual Priority |
|--------|-----------|-----------------|
| Common | +1% | Subtle upgrade |
| Uncommon | +3% | Visible upgrade |
| Rare | +6% | Distinct appearance |
| Epic | +10% | Premium showcase |
| Legendary | +15% | Exclusive showcase |

## Visual Consistency Checklist

- [ ] Brand colors remain consistent across all parts
- [ ] Finish types (matte, gloss, metallic) match brand identity
- [ ] Logo placement visible but not overwhelming
- [ ] Part proportions scale correctly for all scooter models
- [ ] Rarity tier visually distinct (color glows, special effects)
- [ ] Customization combinations feel cohesive
- [ ] No clipping between parts in all poses
- [ ] Materials render consistently across LODs
